# omi-pr

Omi框架的预渲染工具。

## 安装使用 omi-pr

``` js
$ npm install omi-pr -g
```

## 目录结构

![dir](http://images2015.cnblogs.com/blog/105416/201704/105416-20170407165113269-938973687.png)
 
* omi.config.js 为配置文件，在里面配置omi.js的cdn地址
* tpl.html 为所有HTML页面模板文件
* main.js 为页面的入口文件
 
## 编译

在项目根目录下执行omi-pr便可:

``` js
$ omi-pr 
```
